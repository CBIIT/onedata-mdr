When the CDE Browser is retired the CDE links using the legacy formatted URL will temporarily redirect here, to the caDSR II Landing Page. End users will need to select Data Element - View/Browse favorite, re-enter the CDE Public ID into the search filter and select the “View” icon next to the CDE to display details.

In October 2022 caDSR II enabled a Deep Link to replace the legacy URL. This link is visible when viewing CDEs in caDSR II. When implement in customer systems, the Deep Link opens a search result to the specified CDE, skipping the need to re-enter the Public ID.

  • The legacy URL format:

    https://cdebrowser.nci.nih.gov/cdebrowserClient/cdeBrowser.html#/search?publicId=62&version=6.0

  • The new Deep Link URL format:

    https://cadsr.cancer.gov/onedata/dmdirect/NIH/NCI/CO/CDEDD?filter=CDEDD.ITEM_ID=62%20and%20ver_nr=6.0

    The Public ID and Version number in red above are the values that need to be changed when linking to different CDEs or Versions. Version can be entered as n, n.n, or n.nn. If no version number is specified all versions will be returned. Users click the "View" icon next to the desired CDE for the details.

    Example: CDE 62v6.0
