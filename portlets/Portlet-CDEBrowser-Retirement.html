REMINDER -The Legacy CDE Browser Retires Today! Say goodbye to an our steady old friend! If you need help with similar features in caDSR II, please contact the caDSR Registration Authority at the link in the Help Section below!

Please note that CDE Direct links using the legacy formatted URL should be replaced with the new caDSR II Deep Link.

June 1 the CDE Browser will be redirected briefly to a Retirement Notice, then to the caDSR II landing page.

  • Legacy CDE link URL format:

    https://cdebrowser.nci.nih.gov/cdebrowserClient/cdeBrowser.html#/search?publicId=62&version=6.0

  • The new Deep Link URL format:

    https://cadsr.cancer.gov/onedata/dmdirect/NIH/NCI/CO/CDEDD?filter=CDEDD.ITEM_ID=62%20and%20ver_nr=6.0

    For more information regarding how to reformat these URLS, please contact the caDSR RA with questions at caDSR.RA@mail.nih.gov.
