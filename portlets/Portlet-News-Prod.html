caDSR II Release 1.49.1.0 includes the following improvements:

  • VM Match Public Access - This feature allows customers to import a spreadsheet of data values or permitted values for a CDE and match them to an existing Value Meaning (VM) or Concept. See caDSR II User Guides link in the Help Section for access to the template and instructions for use.

  • Retired Content - Improvement have been made to the format of the email message received when users retire content in caDSR that impacts related items.

  • REDCap DD Forms - We made improvements to the mappings between REDCap DD files so that now Question header and footer instructions are included in the exported dictionary.

  • Models and Model Mapping - We are in the final stages of improvements for importing Models for registration in caDSR using a formatted csv file. Once Models are registered with associated CDEs, they can be mapped to other registered models, automatically generating mappings between semantically equivalent fields and semi-automatically between similar fields with different value sets. Please contact the caDR RA for more information.

As always, this release also has a number of UI improvements and bug fixes. For details, refer to the caDSR II Release Notes.

caDSR II provides access to NCI's metadata standards for: Common Data Elements (CDEs), Case Report Forms (CRFs), and Data Models. These metadata resources describe a common way of structuring and understanding data.

The "Links to Favorites" section Data Elements - View/Browse provides access to NCI's CDEs and Forms provides access to our CRFs. It also includes links for browsing or searching specific collections of content using various "Child Filters".

After finding content by searching, you have multiple export options:

  • Export the displayed content using Delivery Options.
  • Add the displayed content to the User Cart, use Download Collection to build a collection (a custom set of content), and request a download of that collection in various formats:
    • Legacy caDSR Excel and XML formats
    • Medidata Rave ALS
    • REDCap Data Dictionaries
