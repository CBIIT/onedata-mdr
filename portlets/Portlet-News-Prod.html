caDSR II Release 1.51.5.0 includes the following improvements:

  • Pediatrics Cancer Core CDEs - This new Favorite displays the CDEs (all statuses) that are being worked on for the Pediatrics Cancer Context.

  • NIH CDE Governance Submission Download - This new format in the Download Collections feature supports exporting caDSR CDEs in a spreadsheet format that can be inserted into the NIH CDE Submission Template. The templates are submitted to the NIH CDE Governance Committee for review and approval for the "NIH Endorsed" status and will then be included in the NIH CDE Repository. CDEs in the NIH CDE Repository are available to search/use within REDCap systems. NCI Standards CDEs will be submitted for endorsement after the first of the year.

  • Models and Model Mapping - Continued improvements to the Beta Model Mapping feature for recording transformation rules that can be used by ETL tool vendors to generate their tool specific programming language implementation. We will be testing this with several ETL Tool vendors over the next several months.

As always, this release also has a number of UI improvements and bug fixes. For details, refer to the caDSR II Release Notes.

caDSR II provides access to NCI's metadata standards for: Common Data Elements (CDEs), Case Report Forms (CRFs), and Data Models. These metadata resources describe a common way of structuring and understanding data.

The "Links to Favorites" section Data Elements - View/Browse provides access to NCI's CDEs and Forms provides access to our CRFs. It also includes links for browsing or searching specific collections of content using various "Child Filters".

After finding content by searching, you have multiple export options:

  • Export the displayed content using Delivery Options.
  • Add the displayed content to the User Cart, use Download Collection to build a collection (a custom set of content), and request a download of that collection in various formats:
    • Legacy caDSR Excel and XML formats
    • Medidata Rave ALS
    • REDCap Data Dictionaries
