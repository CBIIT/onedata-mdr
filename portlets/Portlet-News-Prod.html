caDSR II Release 1.52.3.1 includes the following improvements:

  • CDE REST APIs - We've added additional content to the Query for CDEs by Public ID: Classification Schemes, Reference Documents (Question Text), Alternate Names and Concepts, and fixed an issue retrieving CDEs with non-enumerated Value Domains.

  • Models - View/Browse - This new favorite supports viewing models that have been registered in caDSR. The details of the models are shown, such as Classes, Tables, and Nodes, and the associated Attributes, Columns and Properties. CDE associations are also displayed. As always, the content can be download from any node in the browsing feature using Delivery Options. Remember to "save as" Excel!

  • VM Match Enhancement - Added support for matching PV/Value to external terminologies including ICD-O leveraging the WHO ICD-O files and the ICD-O codes that are mapped to NCIt Concepts via the NCIm/NCIt cross walk file.

caDSR II provides access to NCI's metadata standards for: Common Data Elements (CDEs), Case Report Forms (CRFs), and Data Models. These metadata resources describe a common way of structuring and understanding data.

The "Links to Favorites" section Data Elements - View/Browse provides access to NCI's CDEs and Forms provides access to our CRFs. It also includes links for browsing or searching specific collections of content using various "Child Filters".

After finding content by searching, you have multiple export options:

  • Export the displayed content using Delivery Options.
  • Add the displayed content to the User Cart, use Download Collection to build a collection (a custom set of content), and request a download of that collection in various formats:
    • Legacy caDSR Excel and XML formats
    • Medidata Rave ALS
    • REDCap Data Dictionaries
