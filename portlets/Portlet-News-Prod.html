caDSR II Release 1.47.3.0 includes the following improvements:

  • API Searches - Case Insensitive - Fields like long and short names, Context, Protocol ID and other commonly used fields for searching are mixed case. Case insensitive searching will make it easier.

  • DEC Match - Matches can be targeted to DECs in specific Context(s), helping align with existing models or other Context specific semantics.

  • CDE/DEC Match - Users can now run DEC Match on their imported CDE Match batch without having to reimport/reformat the file, streamlining finding reusable semantic content.

  • VM Match Improvements - User Tips can be imported making it easier to pre-set searching to optimize matches in bulk instead of having to enter user tips one at a time through the front end.

  • Batch Loading - Batch User and Batch Name can be edited to help curators reassign content for review to other individuals and making it easy to find using the built in filters.

  • Model Loading - Beta - Loading models for registration prior to creating CDEs is coming! Contact caDSR RA to request more information!

As always, this release also has a number of UI improvements and bug fixes. For details, refer to the caDSR II Release Notes.

caDSR II provides access to NCI's metadata standards for: Common Data Elements (CDEs), Case Report Forms (CRFs), and Data Models. These metadata resources describe a common way of structuring and understanding data.

The "Links to Favorites" section Data Elements - View/Browse provides access to NCI's CDEs and Forms provides access to our CRFs. It also includes links for browsing or searching specific collections of content using various "Child Filters".

After finding content by searching, you have multiple export options:

  • Export the displayed content using Delivery Options.
  • Add the displayed content to the User Cart, use Download Collection to build a collection (a custom set of content), and request a download of that collection in various formats:
    • Legacy caDSR Excel and XML formats
    • Medidata Rave ALS
    • REDCap Data Dictionaries
