caDSR II Release 1.34.1 went into production July 28 2022. Please see the link to Release Notes.

caDSR provides access to NCI's CDEs and Forms as well as other resources for describing how to collect standardized data.

The first release of caDSR II replaces all prior caDSR web applications except for the CDE Browser and the current caDSR API. We expect to retire those applications over time as stakeholders migrate to this new platform, including migrating to the new caDSR REST APIs. Refer to links in the "Help" section of this page.

The "Links to Favorites" section provides links to tools for searching for CDEs and Forms as well as convenience links for browsing or searching specific collections of content using various "Child Filters".

After finding content by searching, you have multiple export options:

  • Export the displayed content using Delivery Options.
  • Add the displayed content to the User Cart, use Download Collection to build a collection (a custom set of content), and request a download of that collection in various formats:
    • Legacy caDSR Excel and XML formats
    • Medidata Rave ALS
    • REDCap Data Dictionaries
